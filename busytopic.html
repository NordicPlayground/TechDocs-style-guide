    Replacement text¶

    We use the extra feature in the mkdocs.yml file to create replacement shortcut terms frequently used term. This is done by writing a term between two braces like this { { term } } (excluding the extra spaces between the braces).

    { { test } } = This is a test abbreviation snippet
