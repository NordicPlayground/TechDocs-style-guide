# Verbs

Using the present tense is often easier to read and understand than the past or future tense. In the present tense, the action is happening now.

Use either the indicative or imperative form and avoid using the subjunctive form. These are crisp and straightforward without being bossy. Don’t switch moods within a sentence.

| **Mood** | **Use for** | **Examples** |
| -------- | ----------- | ------------ |
| **Indicative**  | Statements of fact, questions, assertions, and explanations. | Style sheets are powerful tools for formatting complex documents. |
| **Imperative**  | Instructions, procedures, direct commands, requests, and headings for columns that list customer actions. | Enter a file name, and then save the file.<br />To do this </br> Select this |
| **Subjunctive** | Wishes, hypotheses, and suggestions— **avoid** . | We recommend that you be careful about opening email attachments. |

## Active and passive voice

Use the active voice whenever you can. In active voice, the subject of the sentence performs the action. In passive voice, the subject is the receiver of the action.

|  **Voice**  |**Examples**|
| ----------- | ---------- |
| **Active**  | Office 365 includes the Office 2016 apps for PC and Mac. |
| **Passive** | When the user clicks **OK**, the transaction is committed. |

## Verb agreement

Verbs have singular and plural forms. Use the verb form that agrees with the subject of the sentence in number.


| **When the subject is** | **The verb is** | **Examples** |
| ----------------------------------------------------- | ------------------------------------------------ | ---------------------------------------------------------------------------------------------------------- |
| A group of things | Singular | A variety of games **is** available from Microsoft Store.|
| Two or more singular things connected by "and" | Plural | Facebook and Twitter **are** available from Microsoft Store.|
| Two or more singular things connected by "or"| Singular| Your tablet or phone **is** all you need to play your favorite games on the go.|
| A singular thing and a plural thing connected by "or" | Singular or plural, to match the closest subject | Skype or social media apps **are** available from Microsoft Store.|

## Phrasal verbs

Phrasal verbs are verb-preposition combinations (like "set up" or "figure out") that function as a single unit. While common in informal English, they often create problems for translation software and non-native English speakers. Use simple, direct verbs instead.

### Why avoid phrasal verbs

Phrasal verbs create difficulties for:

* **Translation tools**: Machine translation often translates each word separately, losing the intended meaning
* **Non-native speakers**: The meaning of phrasal verbs is often not obvious from the individual words
* **Technical accuracy**: Simple verbs are more precise and leave less room for misinterpretation

### Common phrasal verbs and alternatives

Use this table to find simple verb alternatives:

| **Avoid (Phrasal Verb)** | **Use Instead** |
| ------------------------ | --------------- |
| back up | copy, save, duplicate |
| bring up | start, raise, mention, open |
| carry out | perform, complete, execute |
| come up with | create, develop, devise |
| figure out | determine, calculate, understand, discover |
| find out | discover, learn, determine |
| get around | avoid, bypass, circumvent |
| go through | review, examine, inspect |
| leave out | omit, exclude |
| look into | investigate, examine, research |
| make up | compose, constitute, create |
| point out | indicate, note, identify, highlight |
| put together | assemble, compile, create |
| run into | encounter, experience |
| set up | configure, establish, create, initialize |
| shut down | stop, close, terminate, power off |
| start up | start, launch, initiate, boot |
| take apart | disassemble, separate |
| turn off | disable, stop, deactivate, power off |
| turn on | enable, start, activate, power on |
| work around | bypass, avoid, circumvent |

### Examples in Nordic context

**Avoid:**

* "Set up the nRF Connect SDK environment before you start"
* "Figure out which API to use for your application"
* "Turn on the BLE peripheral to start advertising"
* "Look into the error logs to find the problem"

**Use:**

* "Configure the nRF Connect SDK environment before you start"
* "Determine which API to use for your application"
* "Enable the BLE peripheral to start advertising"
* "Examine the error logs to identify the problem"

### When phrasal verbs are acceptable

In code comments, variable names, or when quoting UI text, you may need to use phrasal verbs as they appear in the interface:

* UI element: "Click the **Set Up** button"
* Function name: `system_startup()` (if that's the actual function name)
* Direct quotes: The error states "Unable to set up connection"

However, in your explanatory text, use simple verbs:

* "Click the **Set Up** button to configure your device"
* "The `system_startup()` function initializes the system"
* "This error appears when the system cannot establish a connection"
