# Redundancies

Redundant words and phrases increase reading time without adding information. Remove redundancies to make your writing more concise and direct.

## Redundancy patterns in technical writing

### Redundant modifiers

Remove modifiers that repeat the meaning already contained in the word they modify.

**Examples:**

| Redundant | Concise |
| --------- | ------- |
| completely finished | finished |
| currently pending | pending |
| empty void | void |
| exact same | same |
| forever permanent | permanent |
| most unique | unique |
| previously established | established |
| successfully completed | completed |
| totally destroyed | destroyed |
| very unique | unique |

### Redundant verbs

Some verb phrases contain unnecessary words.

**Examples:**

| Redundant | Concise |
| --------- | ------- |
| boot up | boot |
| compile down | compile |
| connect up | connect |
| print out | print |
| shut down | shut down (acceptable: established term) |
| start up | start |
| test out | test |

### Redundant categories

Don't state the category when it's obvious from the word itself.

**Examples:**

| Redundant | Concise |
| --------- | ------- |
| blue in color | blue |
| 10 MB in size | 10 MB |
| rectangular in shape | rectangular |
| during the month of January | during January |
| in the field of engineering | in engineering |

## Reducing wordiness

### Replace phrases with single words

Many wordy phrases can be replaced by a single word without losing meaning.

| Wordy | Concise |
| ----- | ------- |
| a large number of | many |
| at this point in time | now |
| bring to a conclusion | conclude |
| by means of | by, with |
| give an indication of | indicate |
| has the capability of | can |
| in close proximity to | near |
| is dependent upon | depends on |
| it is necessary that | must |
| make a determination | determine |
| make an adjustment to | adjust |
| on a regular basis | regularly |
| serves as a means of | (rephrase with a verb) |
| take into consideration | consider |

### Tighten common constructions

| Wordy | Concise |
| ----- | ------- |
| There are three modes that support... | Three modes support... |
| It is possible to configure... | You can configure... |
| In order to build the application... | To build the application... |
| It should be noted that... | (omit, or start with the actual point) |
| The fact that the connection failed... | The connection failure... |

## Finding redundancies in your writing

### Read your text aloud

Redundancies that look natural in writing often sound awkward when spoken.

### Ask "Does removing this word change the meaning?"

If removing a word doesn't change the meaning of the sentence, the word is likely redundant.

**Test:**

"Completely eliminate all of the old files."

* Remove "completely" → "Eliminate all of the old files." (same meaning)
* Remove "all of" → "Eliminate the old files." (same meaning)
* Remove "old" → "Eliminate the files." (different meaning — keep "old")

**Result:** "Eliminate the old files."

### Review after writing

Don't edit for redundancy while drafting. Write freely, then review specifically for conciseness.
